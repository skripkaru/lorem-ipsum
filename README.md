# Lorem Ipsum

## Project setup

Install with yarn

```bash
  yarn install
```

Compiles and hot-reloads for development

```bash
  yarn start
```

Lints and fixes scripts

```bash
  yarn lint
```

Lints and fixes styles

```bash
  yarn lint:css
```

Format files

```bash
  yarn format
```

Compiles and minifies for production

```bash
  yarn build
```
